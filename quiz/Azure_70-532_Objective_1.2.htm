Setting up a CNAME record of "awverify.{sitename}.azurewebsites.net" (substitute the "{sitename}" with the name of the Azure Web App) within the DNS of the Custom Domain lets Azure verify that you either own or otherwise have control over the domain name.
This CNAME record must be created on the Custom Domain name prior to setting up the Custom Domain within the Azure Web App configuration.
