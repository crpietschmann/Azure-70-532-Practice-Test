What is contained within the Detailed Error Log?
HTTP transaction information saved in the W3C extended log file format.
Information for HTTP status codes (400 or greater) that indicate a failure, plus additional information that may help determine what caused the server to return the error.
Trace of IIS components used to process requests with the time taken within each component.
Source: http://azure.microsoft.com/en-us/documentation/articles/web-sites-enable-diagnostic-log/
